Hello Boostrap!

This theme installed Bootstrap's 3.3.1 source LESS and Javascript files. It also set up some Grunt watch commands to build CSS from the less folder as you build. You'll need to restart your local server to get it to operate properly.

If you need to update boostrap.

  1. Open your terminal and navigate to this site's directory
  2. Run npm install -g bower
  3. Run bower update
  4. Run grunt copy

This is your homepage, located at /pages/index.html.
